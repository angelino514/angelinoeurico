## 🛒 Projeto E-commerce (E-STORE)

Desenvolvimento de uma aplicação de e-commerce completa utilizando **HTML, CSS e JavaScript puro**.

### ⚙️ Funcionalidades implementadas:
- Listagem dinâmica de produtos
- Sistema de carrinho de compras
- Sistema de favoritos
- Pesquisa de produtos com sugestões automáticas
- Filtro por categorias
- Página de detalhes de produtos
- Banners promocionais dinâmicos
- Gestão de estado com `localStorage`

Este projeto foi desenvolvido com foco em lógica de programação, manipulação do DOM e construção de sistemas reais de interface.
